The context

Before: permanent double entry across 2 systems.

Tool Innovation Panamá is a professional tool distributor based in Central America, with a catalog of several hundred technical products. For management: Odoo (a full ERP - stock, purchasing, accounting). For online sales: a Wix store, already live and taking B2B/B2C orders.

The problem: the two systems didn't talk to each other. The consequences: manual product re-entry (create a product in Odoo, then create it again in Wix with photos, descriptions, prices), permanently out-of-sync stock (selling items online that were physically out of stock), Wix orders never reaching Odoo (booked by hand), and rushed SEO product pages for lack of time.

The solution

A custom connector + AI + dashboard, delivered as an app.

A bidirectional connector built from scratch, packaged as a desktop application installable on Mac and Windows (the client has no server to manage - the app runs on their machine, data flows directly through the Odoo and Wix APIs).

The client launches the app in the morning, sees the sync status, triggers whatever they want manually, or lets the automatic jobs run. Everything is logged, everything is traceable, everything can be undone.

What was delivered

6 key modules, integrated in a single app.

01

Product catalog sync

Odoo → Wix: prices, stock, images, descriptions. New product detection, change updates, deletion handling.

02

Order sync

Wix → Odoo: every Wix order is created automatically in Odoo with the right customer (created if new), the right products, quantities and prices.

03

AI content generation

Automatic SEO product content by Google Gemini (title, meta description, long description) across the hundreds of products in the catalog.

04

Monitoring dashboard

Real-time sync status: latest jobs, metrics (products synced, orders processed), error alerts, full history.

05

Automatic reconciliation

Conflict detection (a product existing in both systems with diverging data), automatic resolution by defined rules, user reporting.

06

Packaged app, Mac & Windows

Native install on the client's machine. No server to manage, no extra hosting costs. Built-in auto-updater.

Detailed tech stack

Under the hood.

Backend: Node.js + TypeScript + Express. Local persistence with SQLite (better-sqlite3). Odoo API over XML-RPC, Wix API over REST + webhooks.

Frontend: React + Vite + Tailwind. Custom design system, multilingual (i18n). Real-time dashboard with smart polling.

AI: Google Gemini API for SEO product content generation. Prompt templates tuned by product type. Local cache to avoid redundant calls.

Packaging: Electron application, distributed as .dmg (Mac) and .exe (Windows) files. Built-in auto-updater.
