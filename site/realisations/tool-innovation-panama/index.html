Le contexte

Avant : double saisie permanente entre 2 systèmes.

Tool Innovation Panama est un distributeur d'outillage professionnel basé en Amérique centrale, avec un catalogue de plusieurs centaines de produits techniques. Côté gestion : Odoo (ERP complet - stock, achats, comptabilité). Côté vente en ligne : boutique Wix, déjà installée et utilisée pour les commandes B2B/B2C.

Le problème : les deux systèmes ne se parlaient pas. Conséquences : ressaisie manuelle des produits (créer un nouveau produit dans Odoo + le créer aussi dans Wix avec photos, descriptions, prix), désynchronisation permanente des stocks (vendre en ligne un produit dont le stock physique est à zéro), commandes Wix qui n'arrivent jamais dans Odoo (à comptabiliser à la main), fiches produit SEO bâclées faute de temps.

La solution

Connecteur sur mesure + IA + dashboard, livré en application.

Connecteur bidirectionnel développé sur mesure, packagé comme une application desktop installable sur Mac et Windows (le client n'a aucun serveur à gérer - l'app tourne sur sa machine, les données passent en direct via les APIs Odoo et Wix).

Le client lance l'app le matin, voit l'état de la sync, déclenche manuellement ce qu'il veut, ou laisse tourner les jobs automatiques. Tout est journalisé, tout est traçable, tout est annulable.

Modules livrés

5 modules clés, intégrés dans une seule app.

01

Sync catalogue produits

Odoo → Wix : prix, stock, images, descriptions. Détection des nouveaux produits, mise à jour des modifications, gestion des suppressions.

02

Sync commandes

Wix → Odoo : chaque commande Wix est créée automatiquement dans Odoo avec le bon client (créé si nouveau), les bons produits, les bonnes quantités, les bons prix.

03

Génération IA des fiches

Génération automatique de fiches produit SEO par Google Gemini (titre, meta description, description longue) pour les centaines de produits du catalogue.

04

Dashboard monitoring

État de la sync en temps réel : derniers jobs exécutés, métriques (produits synchronisés, commandes transitées), alertes en cas d'erreur, historique complet.

05

Réconciliation automatique

Détection de conflits (produit existant dans les 2 systèmes avec données divergentes), résolution automatique selon règles définies, reporting à l'utilisateur.

06

App packagée Mac & Windows

Installation native sur le poste du client. Pas de serveur à gérer, pas de coût d'hébergement supplémentaire. Mise à jour intégrée.

Stack technique détaillée

Sous le capot.

Backend : Node.js + TypeScript + Express. Persistance locale en SQLite (better-sqlite3). API Odoo via XML-RPC, API Wix via REST + webhooks.

Frontend : React + Vite + Tailwind. Design system custom, multilingue (i18n). Dashboard temps réel avec polling intelligent.

IA : Google Gemini API pour la génération de contenu produit SEO. Templates de prompts optimisés par type de produit. Cache local pour éviter les appels redondants.

Packaging : application Electron, distribution via fichiers .dmg (Mac) et .exe (Windows). Auto-updater intégré.
